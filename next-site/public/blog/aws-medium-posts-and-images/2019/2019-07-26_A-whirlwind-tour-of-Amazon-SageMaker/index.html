A whirlwind tour of Amazon SageMaker

Amazon SageMaker was launched at AWS re:Invent 2017, about 18 months ago. In this hour long session, we explain why we built this service in the first place, and then we go over all the new features that have been added since then: security, data labeling, notebook instances, training, and deployment.
