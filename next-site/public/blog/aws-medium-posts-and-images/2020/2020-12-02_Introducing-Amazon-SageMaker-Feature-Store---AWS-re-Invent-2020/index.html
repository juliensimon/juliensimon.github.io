Introducing Amazon SageMaker Feature Store — AWS re:Invent 2020

Amazon SageMaker Feature Store is a new capability of Amazon SageMaker that makes it easy for data scientists and machine learning engineers to securely store, discover and share curated data used in training and prediction workflows.

In this video, I give you a quick tour of Amazon SageMaker Feature Store, a new capability to store your machine learning features for model training and prediction.
