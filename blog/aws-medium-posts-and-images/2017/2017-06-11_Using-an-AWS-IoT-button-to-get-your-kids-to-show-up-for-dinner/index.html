Using an AWS IoT button to get your kids to show up for dinner

The AWS IoT button is now available in Europe. This device is the best way to get started with AWS IoT and build your first project.

In the first video, I’ll show you how to:

  • connect it to the AWS IoT Gateway,
  • configure it,
  • test it using both the AWS IoT Console and a standalone tool called MQTT.fx.

In the second video, you’ll learn how to:

  • create a rule to route IoT messages to a AWS Lambda function,
  • use an SNS topic to send SMS messages,
  • write a Lambda function to send messages based on the click type (single, double, log).

And hopefully, your kids will show up for dinner! Enjoy and have fun :)
