Amazon SageMaker Studio Deep Dive AWS Online Tech Talks

February 27, 2020
Traditional machine learning (ML) development is a complex, expensive, iterative process made even harder because there are no integrated tools for the entire machine learning workflow. You need to stitch together tools and workflows, which is time-consuming and error-prone. Amazon SageMaker Studio solves this challenge by providing all of the components used for machine learning in a single, web-based visual interface. SageMaker Studio gives you complete access, control, and visibility into each step required to build, train, and deploy models. You can quickly upload data, create new notebooks, train and tune models, move back and forth between steps to adjust experiments, compare results, and deploy models to production all in one place, making you much more productive. In this tech talk, we will explain how it works, including a demo.
